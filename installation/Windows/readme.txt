Пошаговая инструкция по установке и запуске бота "из коробки":
1. Проверьте наличие языка программирования NodeJS на вашем компьютере с помощью сочетания клавиш: WIN + R -> cmd -> node -v. Или просто запустите файл "check_node.bat",
он выполнит всё за вас.
2. Если ваша версия NodeJS не v18.12.0, то установите её по ссылке: https://nodejs.org/en/blog/release/v18.12.0
3. Теперь откройте папку с файлами бота (она рядом с папкой installation), потом откройте папку dist и настройте .env файл (за подробностями можете обратиться к разработчику)
4. После настройки файла .env, откройте run.bat

После этих действий бот должен запуститься у вас на компьютере. Если у вас остались вопросы - обращайтесь в Discord (itsme1170) или Telegram (@itsme1170).
